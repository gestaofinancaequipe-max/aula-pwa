# Ícones do PWA

Esta pasta deve conter os ícones do Progressive Web App nas seguintes resoluções:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Gerando os ícones

Você pode usar ferramentas online como:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

Ou criar manualmente uma imagem de 512x512 pixels com tema roxo e usar uma ferramenta para gerar os tamanhos necessários.

**Nota:** Durante o desenvolvimento, o app funcionará sem os ícones, mas para uma instalação completa do PWA, os ícones são recomendados.
